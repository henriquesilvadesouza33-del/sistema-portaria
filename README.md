# sistema-portaria
Sistema de Portaria desenvolvido para controle de entrada e saída de visitantes.  O sistema permite o cadastro de visitantes, registro de acessos e visualização em tempo real das informações, facilitando a organização e segurança do ambiente.  Projeto desenvolvido com foco em usabilidade e interface simples, utilizando HTML, CSS e JavaScript.

# 🏢 Sistema de Portaria

Sistema desenvolvido em Python para controle de entrada e saída de visitantes.

## 🔧 Funcionalidades
- Cadastro de visitantes
- Registro de entrada e saída
- Controle de acesso

## 💻 Tecnologias
- Python
- SQLite

## 🚀 Como executar
1. Instale o Python
2. Execute o comando:
   python app.py

## 📌 Sobre o projeto
Aplicação desenvolvida com foco em organização de dados, usabilidade e simulação de um sistema real de portaria.
